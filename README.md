# DeskNin
DeskNin is a help desk ticket management system built with .NET.

The goal of this project is to provide a simple platform for managing support tickets while also serving as a learning project for building SaaS applications with .NET.

---

## Features

- Ticket creation and management
- Role-based access control (Admin, Technician, User)
- Ticket assignment and status tracking
- Priority management
- Ticket comments and history
- Dashboard overview
