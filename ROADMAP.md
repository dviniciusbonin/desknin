## Roadmap
DeskNin will be developed in three main phases, evolving from a simple ticket management system into a full SaaS platform.

### V1 — Ticket System (MVP)

The first version focuses on the core ticket management functionality.

**Features**

* User authentication (login and logout)
* Initial administrator registration
* Role-based access control:

    * Admin
    * Technician
    * User
* Dashboard with ticket overview
* User CRUD management (admin only)
* Create tickets
* List and view tickets
* Assign a technician to a ticket
* Change ticket status
* Set ticket priority
* Close tickets
* Comments and interaction history

---

### V2 — Multi-Tenant
Transforms the system into a SaaS platform for multiple companies.

**Features**

* Company registration
* Users associated with a company
* Data isolation between companies
* Onboarding flow that creates a company and an admin user
* User management within the company
* Dashboard and tickets filtered by company

---

### V3 — Billing

Adds monetization capabilities to the platform.

**Features**

* Subscription system per company
* Checkout integration for payments
* Access control based on subscription status
* Customer portal for subscription management
* Plan cancellation and updates
* Notifications
